```md
# Feature: Erbjudanden

Source: https://www.vasscompany.se/erbjudanden/

## Scope
- Desktop only
- Swedish only
- Smoke level
- Offer categories as separate requirements
- Contact and newsletter as presence checks only

## Requirements

### R1: Sidans huvudsektion för erbjudanden visas
Sidan `/erbjudanden/` ska visa en tydlig huvudsektion för erbjudanden med svensk sidrubrik.

**Assertion hints:**
- Verifiera att URL innehåller `/erbjudanden/`
- Verifiera att en huvudrubrik (`h1`) finns
- Verifiera att huvudrubriken innehåller text som matchar `Erbjudanden`

### R2: Kategorin Utbildning visas
Erbjudandesidan ska visa erbjudandekategorin **Utbildning**.

**Assertion hints:**
- Verifiera att texten `Utbildning` finns synlig på sidan
- Verifiera att kategorin presenteras som en egen sektion, rubrik, kort eller länk

### R3: Kategorin Föreläsning visas
Erbjudandesidan ska visa erbjudandekategorin **Föreläsning**.

**Assertion hints:**
- Verifiera att texten `Föreläsning` finns synlig på sidan
- Verifiera att kategorin presenteras som en egen sektion, rubrik, kort eller länk

### R4: Kategorin Handledning visas
Erbjudandesidan ska visa erbjudandekategorin **Handledning**.

**Assertion hints:**
- Verifiera att texten `Handledning` finns synlig på sidan
- Verifiera att kategorin presenteras som en egen sektion, rubrik, kort eller länk

### R5: Kontaktsektion finns
Erbjudandesidan ska innehålla någon form av kontaktsektion eller kontaktmöjlighet.

**Assertion hints:**
- Verifiera att text som matchar `Kontakt` eller `Kontakta` finns synlig
- Alternativt verifiera att det finns en länk eller knapp för kontakt
- Presence check only; validera inte funktion eller innehåll i detalj

### R6: Nyhetsbrevssektion finns
Erbjudandesidan ska innehålla någon form av nyhetsbrevssektion eller uppmaning att prenumerera.

**Assertion hints:**
- Verifiera att text som matchar `Nyhetsbrev`, `Prenumerera` eller liknande finns synlig
- Alternativt verifiera att ett formulärfält för e-post i nyhetsbrevskontext finns
- Presence check only; validera inte inskick eller detaljerad funktion
```
